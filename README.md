# UAS Data Science – Campus Recruitment

Studi kasus: **Faktor Akademik yang Mempengaruhi Penempatan Kerja**

## Isi Repo
- `analysis_campus_recruitment.py` → script lengkap (EDA + model + evaluasi + feature importance)
- `analysis_campus_recruitment.ipynb` → notebook versi ringkas
- `figures/` → folder gambar (grafik EDA, confusion matrix, ROC, feature importance)
- `feature_importance_all.csv` → hasil importance semua fitur (RF)
- `Laporan_UAS_Data_Science_CampusRecruitment.docx` → laporan Word

## Cara Menjalankan
```bash
pip install -r requirements.txt
python analysis_campus_recruitment.py
```

## Dataset
Gunakan file: `Campus Recruitment.csv` (disesuaikan path-nya jika beda).
